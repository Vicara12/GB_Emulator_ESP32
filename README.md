
## Hardware components and connections

* **Microcontroller:** ESP32-S3-N16-R8

* **Display:** 2.8 TFT SPI 240x320 v 1.2

* **Buttons:**
  * A       -> GPIO1
  * B       -> GPIO2
  * START  -> GPIO42
  * SELECT -> GPIO41
  * UP     -> GPIO38
  * DOWN   -> GPIO40
  * LEFT   -> GPIO39
  * RIGHT  -> GPIO21
  * EMU    -> GPIO18
